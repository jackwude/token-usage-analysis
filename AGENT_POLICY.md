# AGENT_POLICY.md - 工具/技能调用总则（全局）

## 核心规则（必须遵守）

1. **技能优先**：只要存在**明确匹配**的已安装 Skill，就**优先调用 Skill**完成任务。
2. **不确定先试**：当不确定是否有匹配 Skill 或匹配度不够高时，**先尝试最可能的 Skill**（更快）。
3. **失败即汇报再确认**：若 Skill 失败/不通/不适配（报错、权限、页面变动等），必须：
   - 立刻说明失败点/现象
   - 提供下一步选项：换另一个 Skill 或使用通用工具（browser/web_fetch/exec 等）兜底
   - 由用户确认后再继续（除非用户已明确授权自动兜底）

## 补充原则

- 能不用工具就不用工具（纯思考/写作/总结等可直接完成的任务，不强行调用 Skill）。
- 涉及登录、写入、外发消息、修改配置等高风险动作时，要更谨慎，必要时先二次确认。
