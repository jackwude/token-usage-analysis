## 注意事项

1. **登录状态**: 浏览器必须保持百度网盘登录状态
2. **文件夹选择**: 必须在弹窗中明确选择"影视资源（付想专用）"，不能保存到根目录
3. **大文件转存**: 大文件可能需要较长时间，但转存操作本身是秒完成的
4. **失效链接**: 部分链接可能失效，需要跳过并记录
5. **验证码**: 遇到验证码需要人工介入
6. **标签页管理**: 每个链接转存后关闭标签页，避免打开过多页面

## 常见错误处理

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 保存到根目录 | 未选择目标文件夹就点确定 | 必须先在弹窗中点击"影视资源（付想专用）" |
| 找不到文件夹 | 文件夹名称变化或不存在 | 手动确认文件夹名称，更新配置 |
| 页面加载失败 | 网络问题或链接失效 | 重试或跳过该链接 |
| 验证码拦截 | 百度风控 | 暂停，等待人工处理 |
