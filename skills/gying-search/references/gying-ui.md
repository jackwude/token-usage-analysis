# gying.net 页面要点（供调试/失效时排查）

> 目的：当页面结构变化、搜索结果为空、或提取不到网盘链接时，用这份笔记快速定位。

## 关键页面

- 首页：`https://www.gying.net/`
- 搜索页（当前可用形态）：`https://www.gying.net/s/1---1/<关键词>`
  - 备注：旧写法 `https://www.gying.net/s?q=<关键词>` 近期可能 404/不可用（以实际页面为准）
- 详情页：从搜索结果点击进入（URL 通常形如 `/ac/<id>`）

## 常见交互干扰（很关键）

- **弹窗遮罩层拦截点击**：页面可能出现 `.popup-overlay/.popup-wrapper`，会导致输入框/搜索结果“看得见点不了”。
  - 处理：优先点弹窗右下角 **「14天不再提醒」**；不行再按 `ESC`；再不行点关闭按钮；最后才刷新。
  - 自动化脚本侧：建议在每次关键点击/输入前，检测并移除遮罩层（或用 `page.goto()` 替代 click）。

## 典型信息位置（经验规则）

1. **搜索结果列表**
   - 可能包含「置顶/广告/推荐」项；优先选择标题与关键词最匹配的结果。
   - 若同名多条：优先用户目标类型一致（电影 / 剧 / 动漫）。
   - 同一条结果常有两份链接：**封面链接 + 标题链接**。抽取候选时要先按 `href` 去重。
   - 有些封面链接本身没有文字，但它的 `grandparent/card` 容器里有完整标题、类型、评分；不要因为 `a.textContent` 为空就误判成“无候选”。

2. **豆瓣评分**
   - 详情页可能展示豆瓣评分或豆瓣链接；若没有则标注“暂无评分”。

3. **网盘链接区域**
   - 详情页常见有按网盘来源的 Tab（如：百度网盘/夸克网盘/阿里云盘/迅雷网盘等）。
   - **默认可见分组不一定是目标分组**：有些页面会先落在“迅雷网盘”，此时不能据此判断百度/夸克为空。
   - 正确流程应是：先确认进入了“网盘下载 / 网盘资源”区域，再读取各分组名称与数量，然后切换到目标分组（百度优先，夸克兜底）。
   - 进入对应 Tab 后，提取：
     - 链接（`pan.baidu.com` / `pan.quark.cn`）
     - 更新时间（用于“优先最新”排序）
     - 发布者（若页面给出）

## 失败/风控时的降级策略

- **打不开 gying.net**：尝试刷新/更换网络；仍失败则直接返回“无法访问站点”。
- **百度网盘需要登录/风控**：默认不做逐条有效性检测，标注“未验证有效性”。
- **页面结构变化**：
  1) 先确保拿到了正确的详情页；
  2) 再确保真的进入了“网盘下载 / 网盘资源”区域；
  3) 再确保切到了正确的网盘 Tab（不要停留在默认迅雷分组）；
  4) 最后再抓链接列表。
- **抽取结果为空但页面正文已出现“百度网盘/夸克网盘/pan.baidu.com/pan.quark.cn”**：优先判断为“抽取逻辑未切对层/未切对分组”，不要马上判定无资源。
- **若页面已经显示百度网盘存在（包括分组计数）**：禁止直接写“未找到百度网盘”；必须先做百度二次抽取。
  - 注意一个常见误判点：如果你已经切到了“夸克网盘”分组，再用“当前分组的 DOM”去判断百度是否存在，会误判。
  - 正确做法：进入网盘资源区后，就先基于整页正文/HTML 和分组计数判断百度是否存在；并且即使没点到百度分组，也要先全页扫一遍 `pan.baidu.com` 链接。
  - **切换生效验证**：点击“百度网盘”分组后，用 `a[href*="pan.baidu.com"]` 数量 > 0 来确认切换确实生效；否则多半还停留在其它分组（如夸克/迅雷）。
  - 二次抽取仍失败时，正确表述应是：`页面显示存在百度网盘资源，但当前自动提取未成功`。

## 登录态排查优先级（重要）

如果访问 gying.net 时表现异常，按下面顺序排查：

1. **先确认是否使用了 `gying` 专用 Profile**（`~/.openclaw/chrome-profiles/gying`）
2. 再判断当前页面是否真的落在 `/user/login`
3. 最后才下结论说“专用 Profile 登录态失效”

不要在未确认浏览器上下文的情况下，直接把异常归因为登录失效。很多问题是因为误用了默认浏览器上下文，而不是 `gying` Profile 本身掉登录态。

## 建议的输出字段

每条结果尽量包含：
- 标题
- 豆瓣评分（或“暂无评分”）
- 网盘链接（百度优先；无百度则夸克兜底）
- 提取码（如有）
- 更新时间/发布信息（如有）
