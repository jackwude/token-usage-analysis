# gying.net 页面要点（供调试/失效时排查）

> 目的：当页面结构变化、搜索结果为空、或提取不到网盘链接时，用这份笔记快速定位。

## 关键页面

- 搜索页：`https://www.gying.net/s?q=<关键词>`
- 详情页：从搜索结果点击进入（URL 通常形如 `/ac/<id>`）

## 典型信息位置（经验规则）

1. **搜索结果列表**
   - 可能包含「置顶/广告/推荐」项；优先选择标题与关键词最匹配的结果。
   - 若同名多条：优先电视剧/电影的目标类型一致（用户会在话术里提“电影/剧/动漫”）。

2. **豆瓣评分**
   - 详情页通常会展示豆瓣链接或评分字段；若没有则标注“暂无评分”。

3. **网盘链接区域**
   - 详情页常见有按网盘来源的 Tab（如：百度网盘/阿里云盘等）。
   - 进入「百度网盘」Tab 后，提取：
     - 链接（`pan.baidu.com`）
     - 提取码（若有）
     - 更新时间（用于“优先最新”排序）

## 失败/风控时的降级策略

- **打不开 gying.net**：尝试刷新/更换网络；仍失败则直接返回“无法访问站点”。
- **百度网盘需要登录/风控**：不做有效性检测，标注“未验证有效性”。
- **页面结构变化**：
  1) 先确保拿到了正确的详情页；
  2) 再确保切到了「百度网盘」Tab；
  3) 最后再抓链接列表。

## 登录态排查优先级（重要）

如果访问 gying.net 时表现异常，按下面顺序排查：

1. **先确认是否使用了 `gying` 专用 Profile**（`~/.openclaw/chrome-profiles/gying`）
2. 再判断当前页面是否真的落在 `/user/login`
3. 最后才下结论说“专用 Profile 登录态失效”

不要在未确认浏览器上下文的情况下，直接把异常归因为登录失效。很多问题是因为误用了默认浏览器上下文，而不是 `gying` Profile 本身掉登录态。

## 建议的输出字段

每条结果尽量包含：
- 标题
- 豆瓣评分（或“暂无评分”）
- 百度网盘链接
- 提取码（如有）
- 更新时间/发布来源（如有）
