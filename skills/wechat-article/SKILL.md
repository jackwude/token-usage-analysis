# 微信公众号文章解析

自动解析微信公众号文章，生成结构化总结。

## 触发条件

当用户消息中包含 `mp.weixin.qq.com` 链接时，自动触发此技能。

## 功能

- ✅ 提取文章标题
- ✅ 提取作者/来源
- ✅ 提取正文内容（转 Markdown）
- ✅ 生成结构化总结
- ✅ 绕过微信反爬（伪装 iOS 微信 App）

## 使用方式

**用户只需直接转发微信文章链接**，无需任何指令。

示例：
```
https://mp.weixin.qq.com/s/xxxxxxxxxx
```

## 底层实现

调用脚本：`~/.openclaw/workspace/scripts/wechat-article.py`

依赖：
- `httpx` — HTTP 请求
- `beautifulsoup4` — HTML 解析
- `markdownify` — 转 Markdown

## 输出格式

```
📰 标题：《文章标题》
📝 来源：作者/公众号
🔗 链接：https://...

━━━━━━━━━━━━━━━━━━

## 内容摘要

（正文前 3000 字，Markdown 格式）

━━━━━━━━━━━━━━━━━━

💡 需要我总结重点或提取更多信息吗？
```

## 限制

- ❌ 付费文章/粉丝可见文章可能无法解析
- ❌ 图片仅保留占位符，无法下载
- ❌ 遇到验证页面会提示需要人工验证

## 故障排除

如果解析失败：
1. 检查链接是否有效（未过期）
2. 确认不是付费/限制访问文章
3. 尝试让用户复制正文内容直接发送
