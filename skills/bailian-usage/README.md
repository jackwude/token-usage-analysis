# 百炼套餐用量查询 Skill

## 安装

本 Skill 已内置于 workspace，无需额外安装。

## 使用方法

### 快捷命令

直接发送以下任一消息即可触发：

- `查百炼额度`
- `查 Token`
- `Token 用量`
- `看看阿里云还剩多少额度`
- `百炼用量情况`
- `查一下百炼套餐用量`

### 手动调用

告诉 AI："调用 bailian-usage Skill 查询百炼套餐"

## 输出示例

```
## 📊 百炼 Coding Plan 套餐详情

**套餐基本信息：**
- 套餐名称：Lite 基础套餐
- 套餐状态：✅ 生效中
- 剩余天数：28 天
- ...
```

## 登录方式

### 方案 A：Profile 持久化（推荐 ✅）

首次配置：
1. 告诉 AI："配置阿里云百炼的 Profile 登录"
2. AI 会打开浏览器，你手动登录一次
3. 登录后状态自动保存，以后无需再登录

**优势**：一次登录，长期有效，支持 HttpOnly Cookies

### 方案 B：账号密码（备选）

在 `TOOLS.md` 中配置账号密码（见上方账号信息）

## 注意事项

1. **优先使用 Profile 方案** - 更高效稳定
2. **查询完成后会自动关闭浏览器页面**（遵循激进省内存策略）
3. 用量数据以百炼控制台实时显示为准
4. Profile 失效时会自动切换到账号密码方案
