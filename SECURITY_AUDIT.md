## 📋 权限范围

### 读取权限
- `~/.openclaw/agents/*/sessions/*.jsonl` - 仅读取 usage 字段（Token 统计）
- **不读取**：对话内容、消息正文、用户输入

### 写入权限
- `~/.openclaw/logs/session-usage.log` - 用量日志
- `~/.openclaw/logs/token-usage-collector.log` - 收集器运行日志

### 执行权限
- 定时任务（每小时执行 1 次）
- 仅执行本地 Python 脚本

---

## 🔍 数据流分析

### 收集流程
```
Session 文件 (jsonl)
    ↓
提取 usage 字段 (input/output/total)
    ↓
写入日志文件 (session-usage.log)
    ↓
分析报告 (终端输出)
```

### 不包含的数据
- ❌ 对话内容
- ❌ 用户输入/输出文本
- ❌ API Key 或认证信息
- ❌ 模型参数或配置
- ❌ 网络请求或外部调用

---

## 🛡️ 安全设计

### 1. 最小权限原则
- 只读取必要的数据（usage 字段）
- 只写入指定的日志文件
- 不访问网络

### 2. 数据隔离
- 日志文件仅包含统计数据
- 原始 session 文件不被修改
- 无数据外发

### 3. 透明可审计
- 所有代码开源可读
- 日志文件格式清晰
- 定时任务配置透明

## ⚠️ 用户须知

### 日志文件内容
```

```

包含：
- ✅ Session ID（UUID）
- ✅ Agent 名称
